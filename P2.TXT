
2. Insurance Data Modernization (Azure)

Environment: Azure Data Factory, Azure Databricks, ADLS Gen2, Azure Synapse, Hive, Spark, PySpark
Duration: Jan 2020 – Mar 2022
Description: Migrated on-premises insurance customer and payment systems to Azure cloud using modern
 Big Data architecture and ETL pipelines.
Roles & Responsibilities
•	Developed end-to-end ETL pipelines using Azure Data Factory and Azure Databricks.
•	Processed large-scale insurance data using Spark and PySpark.
•	Migrated on-premise Cassandra and Mainframe data into ADLS Gen2.
•	Created Hive tables and optimized queries for reporting requirements.
•	Integrated Azure Synapse Analytics for enterprise reporting and analytics.
•	Implemented data quality validations and transformation logic using PySpark.
•	Worked on Delta Lake architecture for optimized data storage and faster querying.
•	Collaborated with business teams to gather requirements and deliver technical solutions.
